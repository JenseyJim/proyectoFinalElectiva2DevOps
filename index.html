Práctica Final DevOps CI/CD

Bienvenido. Esta página muestra el resultado de una práctica final del curso de DevOps. Además, se configuró GitHub Actions para ejecutar pruebas automáticamente con cada cambio, y se publicó el sitio en GitHub Pages.
